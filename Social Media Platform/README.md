# 🌐 ConnectHub - Full Stack Social Media Platform

 A modern social networking platform built with HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

---

## 🚀 Overview

ConnectHub is a full-stack social media platform that allows users to create profiles, share posts, interact through likes and comments, follow other users, and receive notifications.

The project follows a professional frontend-backend architecture and demonstrates real-world web development concepts such as authentication, API design, user interactions, and responsive UI development.

---

## ✨ Key Features

### 👤 User Authentication

* User Registration
* Secure Login System
* JWT Authentication
* Protected Routes

### 📝 Posts Management

* Create Posts
* View Feed
* Delete Posts
* User Post History

### ❤️ Social Interactions

* Like Posts
* Comment on Posts
* Follow Users
* Unfollow Users

### 🔔 Notification System

* Follow Notifications
* Like Notifications
* Comment Notifications

### 👨‍💼 User Profile

* View Profile
* Edit Profile
* Profile Picture Support
* Activity Tracking

### 📱 Responsive Design

* Mobile Friendly
* Tablet Compatible
* Desktop Optimized
